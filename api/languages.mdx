---
title: "languages"
description: "The languages module lets extensions register LSP feature providers — completion, hover, definition, formatting, code actions, and more."
---

The `languages` module is the primary extension point for contributing language intelligence features. Access it via:

```javascript
const { languages } = require('coc.nvim')
```

All registration methods return a `Disposable`. Push it to `context.subscriptions` so coc.nvim automatically unregisters the provider when the extension is deactivated.

---

## Document selectors

Most methods accept a `DocumentSelector` as their first argument. This controls which documents the provider applies to.

```typescript
// Match by language ID
'typescript'

// Match by scheme and pattern
{ scheme: 'file', pattern: '**/*.ts' }

// Array of selectors (OR logic)
[{ language: 'typescript' }, { language: 'javascript' }]
```

## Completion

### registerCompletionItemProvider

```typescript
registerCompletionItemProvider(
  name: string,
  shortcut: string,
  selector: DocumentSelector | string | null,
  provider: CompletionItemProvider,
  triggerCharacters?: string[],
  priority?: number,
  allCommitCharacters?: string[]
): Disposable
```

Registers a completion provider. `name` is the unique source name shown in logs; `shortcut` is the abbreviated label shown in the popup menu (e.g., `'TS'`).

The provider must implement `provideCompletionItems(document, position, token, context)` and optionally `resolveCompletionItem(item, token)`.

```javascript
const disposable = languages.registerCompletionItemProvider(
  'myext-completion',
  'MY',
  ['typescript', 'javascript'],
  {
    provideCompletionItems(document, position, token, context) {
      return [
        { label: 'myKeyword', kind: CompletionItemKind.Keyword }
      ]
    }
  },
  ['.', '"']  // trigger characters
)
context.subscriptions.push(disposable)
```

---

### registerInlineCompletionItemProvider

```typescript
registerInlineCompletionItemProvider(
  selector: DocumentSelector,
  provider: InlineCompletionItemProvider
): Disposable
```

Registers a provider for inline ghost-text completions (like Copilot). The provider implements `provideInlineCompletionItems(document, position, context, token)`.

## Hover

### registerHoverProvider

```typescript
registerHoverProvider(selector: DocumentSelector, provider: HoverProvider): Disposable
```

Registers a hover provider. The provider implements `provideHover(document, position, token)` and returns a `Hover` object with markdown content.

```javascript
languages.registerHoverProvider(['typescript'], {
  provideHover(document, position, token) {
    return {
      contents: { kind: 'markdown', value: '**My hover content**' }
    }
  }
})
```

## Formatting

### registerDocumentFormattingEditProvider

```typescript
registerDocumentFormattingEditProvider(
  selector: DocumentSelector,
  provider: DocumentFormattingEditProvider,
  priority?: number
): Disposable
```

Alias of `registerDocumentFormatProvider`. Registers a full-document formatter. The provider implements `provideDocumentFormattingEdits(document, options, token)`.

```javascript
languages.registerDocumentFormattingEditProvider(['json'], {
  provideDocumentFormattingEdits(document, options, token) {
    const text = JSON.stringify(JSON.parse(document.getText()), null, options.tabSize)
    const range = Range.create(0, 0, document.lineCount, 0)
    return [TextEdit.replace(range, text + '\n')]
  }
})
```

## Code Actions

### registerCodeActionProvider

```typescript
registerCodeActionProvider(
  selector: DocumentSelector,
  provider: CodeActionProvider,
  clientId: string | undefined,
  codeActionKinds?: CodeActionKind[]
): Disposable
```

Registers a code action provider. `clientId` identifies the source (e.g., the language client ID). `codeActionKinds` restricts the kinds this provider handles (e.g., `[CodeActionKind.QuickFix]`).

The provider implements `provideCodeActions(document, range, context, token)` and optionally `resolveCodeAction(action, token)`.

```javascript
languages.registerCodeActionProvider(
  ['typescript'],
  {
    provideCodeActions(document, range, context, token) {
      return [{
        title: 'My quick fix',
        kind: CodeActionKind.QuickFix,
        edit: { /* WorkspaceEdit */ }
      }]
    }
  },
  'my-ext-client',
  [CodeActionKind.QuickFix]
)
```

```typescript

ProviderName.Definition      // 'definition'
ProviderName.CodeAction      // 'codeAction'
ProviderName.Format          // 'format'

ProviderName.SemanticTokens  // 'semanticTokens'

// …and more
```

## Full example

```javascript
const { languages, workspace, Disposable, CompletionItemKind } = require('coc.nvim')

async function activate(context) {
  // Completion
  context.subscriptions.push(
    languages.registerCompletionItemProvider(
      'myext',
      'MY',
      ['plaintext'],
      {
        provideCompletionItems() {
          return [{ label: 'hello', kind: CompletionItemKind.Text }]
        }
      }
    )
  )

  // Hover
  context.subscriptions.push(
    languages.registerHoverProvider(['plaintext'], {
      provideHover(document, position) {
        return { contents: ['Hover from myext'] }
      }
    })
  )
}
```
